# 🏨 LuxeStay - Modern Hotel Management System

LuxeStay ek professional-grade web application hai jo hotel rooms ko manage aur book karne ke liye banaya gaya hai. Ye project modern tech stack (React, Redux Toolkit, Firebase, Tailwind CSS) ka use karke ek premium user experience provide karta hai.

## ✨ Key Features

- **User Authentication:** Firebase Auth ka use karke secure Sign-In aur Sign-Up process.
- **Dynamic Room Listing:** Firestore Database se real-time rooms fetch aur display karna.
- **Filtering & Sorting:** Room type (Deluxe, Suite, etc.) ke hisaab se filter aur price ke hisaab se sorting (High/Low).
- **Advanced UI/UX:** Tailwind CSS v3 ka use karke modern Glassmorphism aur responsive design.
- **State Management:** Redux Toolkit ka use karke efficient global state management.
- **Reservation System:** Authenticated users ke liye room booking functionality.

## 🛠️ Tech Stack

- **Frontend:** React.js (Vite)
- **Styling:** Tailwind CSS v3
- **State Management:** Redux Toolkit (RTK)
- **Backend/Database:** Firebase Firestore
- **Authentication:** Firebase Auth
- **Routing:** React Router DOM v6
